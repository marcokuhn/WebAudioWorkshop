What is Flexbox?

Flexbox is a one-dimensional layout method for arranging items in rows or columns. It's perfect for creating responsive audio interfaces that adapt to different screen sizes.

Key Concepts:

  • Flex Container: The parent element with display: flex
  • Flex Items: Direct children of the flex container
  • Main Axis: Primary direction (row or column)
  • Cross Axis: Perpendicular to the main axis

Flexbox Cheat Sheet

Container Properties

  • display: flex;
  • flex-direction: row | column;
  • justify-content: flex-start | center | space-between;
  • align-items: flex-start | center | stretch;
  • gap: 20px;
  • flex-wrap: wrap | nowrap;

Item Properties

  • flex: 1; (grow to fill space)
  • flex-shrink: 0; (don't shrink)
  • flex-basis: 200px; (base size)
  • align-self: center; (override container)
  • order: 1; (change visual order)
