HTML Structure Basics

Understanding the fundamental building blocks of web pages

Semantic HTML | Containers | Forms

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses <tags> to structure content.

Semantic HTML Elements

Semantic elements clearly describe their meaning to both the browser and the developer.

Why Use Semantic HTML?

  • Accessibility: Screen readers understand the structure
  • SEO: Search engines better understand your content
  • Maintainability: Code is easier to read and understand
  • Consistency: Standard structure across projects

Common Semantic Elements

<header>
Introductory content or navigation
<nav>
Navigation links
<main>
Main content of the document
<section>
Thematic grouping of content
<article>
Self-contained composition
<aside>
Content aside from the main content
<footer>
Footer information

Container Elements

Containers are used to group and organize content. The most common are <div> (block-level) and <span> (inline).

Form Elements for Audio Controls

Forms and input elements are essential for interactive audio applications. Here are the most common types:

Interactive Elements

Button

<button>Click Me</button>

Range Slider (for frequency, volume, etc.)

440 Hz
<input type="range" min="20" max="2000" value="440">

Number Input


<input type="number" min="60" max="200" value="120">

Select Dropdown (for waveform selection, etc.)


<select><option>...</option></select>

Checkbox (for on/off toggles)


<input type="checkbox" id="muteCheck">

Radio Buttons (for exclusive choices)


<input type="radio" name="channels">

Text Input


<input type="text" placeholder="...">

HTML Attributes

Attributes provide additional information about HTML elements:

Common Attributes

  • id - Unique identifier (e.g., id="playButton")
  • class - One or more class names for styling (e.g., class="btn primary")
  • style - Inline CSS styling (e.g., style="color: red;")
  • data-* - Custom data attributes (e.g., data-frequency="440")
  • onclick - JavaScript event handler
  • disabled - Disables an element
  • placeholder - Hint text for inputs

Example with Multiple Attributes:



<button id="audioBtn" class="btn primary" data-frequency="440" style="..."> Play A440 </button>

Key Takeaways

  • HTML provides the structure of web pages using tags
  • Semantic elements (<header>, <main>, <footer>) improve accessibility and SEO
  • Containers (<div>, <span>) group and organize content
  • Form elements (<input>, <button>, <select>) enable interactivity
  • Attributes provide additional information and functionality
