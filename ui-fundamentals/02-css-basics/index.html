CSS Basics for Audio Applications

Learn how to style your web audio interfaces

CSS Selectors

Selectors determine which HTML elements to style.

Element Selector

Styles all elements of a type

p { color: #333; font-size: 16px; }

Class Selector

Styles elements with a specific class

.example-text { color: #667eea; font-weight: bold; }

ID Selector

Styles a single unique element

#unique-paragraph { background-color: #fff3e0; padding: 10px; }

Multiple Classes

Element can have multiple classes

<p class="example-text large-text">...</p>

Positioning

Static (default)
Relative (shifted from normal position)

This is the parent container
.element { position: static; /* Default, normal flow */ position: relative; /* Offset from normal position */ position: absolute; /* Positioned relative to parent */ position: fixed; /* Positioned relative to viewport */ position: sticky; /* Switches between relative and fixed */ }

Key Takeaways

  • CSS selectors target specific HTML elements for styling
  • The box model controls spacing with margin, padding, and border
  • Colors can be defined using hex, RGB, RGBA, or gradients
  • Typography properties control text appearance
  • Display properties control layout behavior
  • Transitions create smooth animations on hover and other events
  • Custom styling transforms basic inputs into professional interfaces
