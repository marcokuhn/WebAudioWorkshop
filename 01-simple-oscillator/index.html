What's happening?

Tone.js creates an oscillator that generates a continuous waveform at the frequency you specify. The frequency (measured in Hertz/Hz) determines the pitch:

  • Lower frequency (50-200 Hz) = Bass/low notes
  • Middle frequency (200-800 Hz) = Mid-range notes
  • Higher frequency (800-2000 Hz) = Treble/high notes

Different waveforms produce different timbres (tone colors):

  • Sine - Pure tone, fundamental frequency only
  • Square - Hollow sound, odd harmonics
  • Sawtooth - Bright sound, all harmonics
  • Triangle - Softer than sawtooth, odd harmonics

Key Concepts:

  • Tone.Oscillator generates continuous audio signals
  • .toDestination() connects the oscillator to your speakers
  • .start() begins audio generation
  • .stop() halts audio generation
  • Frequency is measured in Hertz (cycles per second)
  • Must start audio context after user interaction (browser security)

Basic Code Example:

// Create an oscillator
const oscillator = new Tone.Oscillator({
    frequency: 440,    // Frequency in Hz (A4 note)
    type: 'sine'       // Waveform type
}).toDestination();    // Connect to speakers

// Start the oscillator
oscillator.start();

// Stop the oscillator
oscillator.stop();
